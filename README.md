# 🤖 AI智能伴侣 (AI Partner)

基于 Streamlit 和 DeepSeek API 构建的智能对话应用，让你拥有一个可定制性格的AI伴侣。
## ✨ 功能特性

- 💬 **智能对话**：集成 DeepSeek 大模型，支持自然语言交互
- 🎭 **自定义性格**：自由设置伴侣昵称和性格特征
- 💾 **会话管理**：支持新建、加载、删除多个会话
- 📜 **历史保存**：会话记录自动保存到本地 JSON 文件
- ⚡ **流式输出**：实时显示AI回复，无需等待完整生成
-  **响应式界面**：Streamlit 构建的美观交互界面

## 🚀 快速开始

### 环境要求

- Python 3.8 或更高版本
- DeepSeek API Key（前往 [DeepSeek官网](https://platform.deepseek.com/) 获取）

### 1. 安装依赖

```
bash
pip install streamlit openai requests
```
### 2. 配置 API Key

在代码 **第 123-125 行** 配置 DeepSeek API Key：

```
python
# 方式一：通过环境变量配置（推荐）
# Windows CMD:
set DEEPSEEK_API_KEY=your_api_key_here

# Windows PowerShell:
$env:DEEPSEEK_API_KEY="your_api_key_here"

# Linux/Mac:
export DEEPSEEK_API_KEY=your_api_key_here
python
# 方式二：直接修改代码（仅测试用）
# 第123行修改为：
client = OpenAI(
    api_key='sk-your-actual-api-key-here',  # 替换为你的API Key
    base_url="https://api.deepseek.com"
)
```
### 3. 运行应用

```
bash
streamlit run ai_partner.py
```
应用启动后会自动在浏览器中打开，默认地址：`http://localhost:8501`

## 📖 使用说明

### 基本操作

1. **设置伴侣信息**
   - 在左侧边栏输入昵称（默认：喜洋洋）
   - 设置性格描述（默认：聪明机智的小男孩）

2. **开始对话**
   - 在底部输入框输入消息
   - 按回车或点击发送按钮
   - AI会以设定的性格回复你

3. **会话管理**
   - **新建会话**：点击侧边栏 "✒️ 新建会话" 按钮
   - **加载历史**：点击 "📘 历史会话" 列表中的会话
   - **删除会话**：点击会话右侧的 "❌" 按钮

4. **查看会话信息**
   - 页面顶部显示当前会话名称
   - 格式：`年-月-日_时-分-秒`（如：2026-04-06_15-30-22）

### 高级技巧

- 修改伴侣性格后，新对话会立即生效
- 会话文件保存在 `sessions/` 目录下
- 每次新建会话前会自动保存当前会话
- 支持流式输出，可实时看到AI逐字回复

## 🔧 代码说明

### 核心功能模块

- **会话管理**（第 20-78 行）
  - `save_session()`：保存当前会话到 JSON 文件
  - `load_sessions()`：加载所有历史会话列表
  - `load_session()`：加载指定会话
  - `delete_session()`：删除指定会话

- **AI对话**（第 177-214 行）
  - 支持流式输出（`stream=True`）
  - 自动维护对话上下文
  - 实时显示AI回复内容

- **系统提示词**（第 84-98 行）
  - 定义AI的角色和行为规则
  - 动态注入昵称和性格参数
  - 确保回复符合伴侣设定

## ⚠️ 注意事项

1. **API Key 安全**
   - ⚠️ 不要将 API Key 硬编码在代码中上传到公共仓库
   - ✅ 推荐使用环境变量方式配置
   - 建议将 `.env` 文件添加到 `.gitignore`

2. **数据存储**
   - 会话数据保存在本地 `sessions/` 目录
   - 每个会话为独立的 JSON 文件
   - 删除仓库不会影响已保存的会话

3. **网络要求**
   - 需要稳定的网络连接访问 DeepSeek API
   - 首次加载可能需要几秒钟响应时间

##  常见问题

**Q: 为什么AI没有回复？**
- 检查 API Key 是否正确配置
- 确认网络连接正常
- 查看控制台是否有错误信息

**Q: 如何修改AI的性格？**
- 在左侧边栏修改"性格"输入框内容
- 修改后立即生效，影响后续对话

**Q: 会话保存到哪里了？**
- 保存在项目目录下的 `sessions/` 文件夹
- 文件名格式：`年-月-日_时-分-秒.json`
